Código-fonte

O comnado document.getElementById('teste') é escrito em linguagem de programação JavaScript, o problema dessa exibição é que fica confuso pelo espaçamento das letras, coisa que não acontece no código literal, para sanar esse problema usamos a tag <code> que organizará melhor os espaçamentos e indicará ao navegador que se trata deum fragmento de um Código-fonte, e ficará assim, document.getElementById('teste')

Outro exemplo:

num = int(input('Digite um número: ')) if num % 2 == 0; print(f'O número {num} é PAR.') else: print(f'O número {num} é IMPAR.'') print('Fim do programa')

Dessa forma resolvemos o problema da legibilidade, mas temos outro problema que seria as indentações do código que pode ser resolvido com outra tag que conterá a tag <code>, a tag <pre>.

        
            num = int(input('Digite um número: '))

            if num % 2 == 0;
                print(f'O número {num} é PAR.')
            else:
                print(f'O número {num} é IMPAR.'')

            print('Fim do programa')

Citações simples

Para se destacar algum texto que foi citado por outra pessoa ou um trecho de um livro ou outro texto que não seja de sua autoria podemos usar a tag <q> dessa forma o navegador e os dispositivos de busca entenderão que se trata de uma citação dando valor semântico ao trecho como no exemplo: O computador é um burro muito rápido..., frase dita pelo pai de um instrutor de informática

Citações completas

No caso de citar um livro ou similia usamos a tag <blockquote> com o parâmentro 'cite="xxxx"'que conterá o endereço web que o livro está, dessa forma a sintaxe ficará assim: <blockquote cite="https//www.endereço_do_livro.com/outros_parametros/">, exemplo da tag sendo usada.

Abreviações

Quando queremos inserir uma legenda sobre uma sigla utilizamos a tag <abbr>, assim toda vez que o usuário posicionar o ponteiro do mouse sobre a sigla aparecerá o significado da sigla.
