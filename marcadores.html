Marcadores de texto

Texto marcado

Podemos criar também um texto marcado usando a tag <mark> que marca o texto como um marca texto comum, a cor da marcação deve ser modificado pelo CSS

Para personalizar a cor da marcação do texto usamos a seguinte sintaxe <mark style="background-color:'cor';, que recebe o nome CSS inline que será estudado mais tarde.

Tamanho de texto

Utilizando a tag <big> podemos aumentar o tamanho de um texto porém esse recurso está obsoleto.

Para diminuir um texto podemos usar a tag <small> que apesar de ser também antiga não está caindo em desuso, pois ainda tem valor semântico por exemplo letras miúdas de um contrato.

Texto deletado / Texto inserido

Podemos marcar um texto que foi corrigido ou retirado como alguma parte da lei como texto deletado, assim o texto aparecerá com um traço continuo ao meio.

Para isso usamos a tag <del>.

Para textos inseridos utilizamos a tag <ins> com isso o tetxo fica sublinhado assim o texto ganha significado de inserido posteriormente, para sublinhar um texto ou palavra podemos usar também a tag <u>, porém sem valor semântico.

Tetxo sobre-escrito / Texto sub-escrito

Quando existe a necessidade de se adicionar um expoente ou algo semelhante utilizamos a tag <sup> que deixará a parte marcada com fonte menor e acima da linha do texto, como exemplo uma equação x2+3x+2=0

Para o resultado oposto temos a tag <sub> que também deixará a fonte menor porém alinha abaixo do texto como em H2O
