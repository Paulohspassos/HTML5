Inserindo vídeos localmente

Outro tipo de mídia muito importante em sites nos dias de hoje são os vídeos.

Para inserir video podemos usar a tag <video src="pasta/nome_do_video.formato">

Este video está hospedado no meu próprio servidor.

Para dimencionar o vídeo devemos usar o parâmetro 'width="000"', e para inserir controles usamos o parâmetro "controls"

Outro parâmentro que é bom termos conhecimento é o 'poster="local_da_imagem.formato"', esse parâmetro irá inserir uma imagem padrão ao nosso video.

Para se inserir outros formatos do arquivo usamos a tag <source> sempre identificando o 'type="tipo_de_arquivo/formato"' para que todos os navegadores consigam abrir o arquivo que disponibilizaremos ao usuário, e sempre observar a ordem que os arquivos estão dispostos na pasta referênte ao arquivo.

O problema de se utilizar um arquivo tão grande como um video hospedado em nosso prórpio servidor, é o gasto com tráfego, pode causar gastos financeiros para o cliente que está hospedando o arquivo.

Podemos ter o vídeo hospedado no servidor onde a página está hospedada, porém a forma mais recomendada é no youtube.

Incorporação de videos

Para inserir um vídeo que está hospedado em outro local como o YouTube devemos inserir usando iframe bastando copiar o embeded direto da página que está hospedado.
