Tabelas em HTML5

Autilização de tabelas em páginas html já foi mais frequente no passado porém ainda é um recurso utilizavel para organizar conteúdos sem utilizar CSS, claro que sempre é interessante essa organização pelo CSS ou outros como queery, mas como o recurso ainda é válido, vale a pena obter esse conhecimento.

NUNCA UTILIZE TABELAS PARA CRIAR A ESTRUTURA DO NOSSO SITE!!!

Uso correto

As tabelas são extremamente importantes e utilizados em sites principalmente aqueles que precisa apresentar dados como listas ou estatisticas entre outros.

A anatomia basica de uma tabela é constituído por linhas e colunas (rows and columns) e para se inserir ao corpo do nosso site basta inseri-los através da tag <table>, lembrando que as tabelas mais simples têm a seguinte hierarquia:

  • TABLE = tabela
    • TABLE ROW = linha de tabela
      • TABLE HEADER = cabeçalho de tabela
      • TABLE DATA = dado de tabela

Agora criando a tabela inserimos a tag "table" e dentro da abertura e fechamento dessa tag inserimos outra tag que é a <tr> que é a linha da nossa tabela, dentro da abertura e fechamento da tag "tr" inserimos a tag <td> que é a celula, ou seja vamos inserir quantas celular necessitarmos cada uma em uma coluna, para inserir mais linhas basta inserirmais tags "tr".

A1 B1 C1
A2 B2 C2
A3 B3 C3
A4 B4 C4

Para se inserir bordas à nossa tabela devemos inserir bordas à nossa tabela, para isso deveremos usar o seguinte código.

        
            table {
                width: 200px;
                border-collapse: collapse;
            }
    
            .linha {
                background-color: lightgrey;
            }
    
            td {
                border: 1px solid black;
                padding: 5px;
                text-align: center;
            }

Para que as linhas da nossa tabela fiquem únicas e não duplas devemos utilizar a configuração "border-collapse: collapse;", por padrão vem com o valor "separate".

O fechamento das tags tr e td são opcionais.

Para alinha verticalmente o conteúdo da nossa tabela podemos usar a configuração "vertical-align:" com os valores top, middle ou bottom
