Colunas em tabelas

Quando houver a necessidade de destacar colunas em tabelas deveremos usar a tag <colgroup%>, nela vamos inserir tags <col> cada tag "col" será indexada a uma coluna assim podemos inserir de forma automática classes, ao fazermos isso pode selecionar uma coluna inteira apenas com essa configuração.

Cadastro
Nome sexo Idade Profissão
Carla Rejane F 34 Enfermeira
Paulo Henrique M 47 Programador
Maria Geralda F 66 Aposentada
Rodrigo Candido M 41 Técnico de Manutenção
Luciano Nunes M 38 Empresário

O código para esse recurso é o seguinte.

        
            HTML 
            <table>
                <colgroup>
                    <col class="cnome">
                    <col class="csexo">
                    <col class="cidade">
                    <col class="cprof">
                </colgroup>
                <caption>Cadastro</caption>
                <tr>
                    <th scope="col">Nome</th>
                    <th scope="col">sexo</th>
                    <th scope="col">Idade</th>
                    <th scope="col">Profissão</th>
                </tr>
                <tr>
                    <td>Carla Rejane</td>
                    <td>F</td>
                    <td>34</td>
                    <td>Enfermeira</td>
                </tr>
                <tr>
                    <td>Paulo Henrique</td>
                    <td>M</td>
                    <td>47</td>
                    <td>Programador</td>
                </tr>
                <tr>
                    <td>Maria Geralda</td>
                    <td>F</td>
                    <td>66</td>
                    <td>Aposentada</td>
                </tr>
                <tr>
                    <td>Rodrigo Candido</td>
                    <td>M</td>
                    <td>41</td>
                    <td>Técnico de Manutenção</td>
                </tr>
                <tr>
                    <td>Luciano Nunes</td>
                    <td>M</td>
                    <td>38</td>
                    <td>Empresário</td>
                </tr>
            </table>

            CSS 

            <style>
                body {
                    font-family: sans-serif;
                }
                table {
                    border-collapse: collapse;
                    width: 500px;
                }
                th, td {
                    border: 1px solid black;
                    padding: 8px;
                }
                .cnome {
                    background-color: chartreuse;
                }
            </style>
