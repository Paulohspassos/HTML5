Trabalhando com Listas

Um elemento muito importante em web sites são as listas, elas anumeram e relacionam outros elementos na pagina.

Listas Ordenadas OL

Para se obter uma lista ordenada por algum marcador usamos a tag <ol> ela adicionará uma marca como um número ou algarismo romano ou letra para ordenar essa lista. Listas ordenadas são aquelas que a ordem dos itens faz diferênça.

Para se inserir os itens da lista, usamos a tag <li> "list item".

  1. Acordar
  2. Tomar café
  3. Arrumar ambiente de estudos
  4. Estudar
  5. Praticar conteúdo estudado

Para alterar o marcador utilizamos o parâmetro 'type="X"'.

  1. Acordar
  2. Tomar café
  3. Arrumar ambiente de estudos
  4. Estudar
  5. Praticar conteúdo estudado

Para se iniciar a lista em um valor diferente usamos o parãmentro 'start="X"'.

  1. Preparar almoço
  2. Arrumar cozinha
  3. Estudar
  4. Praticar conteúdo estudado
  5. Preparar café da tarde
  6. Receber minha esposa AMADA

Listas Não Ordenadas UL

Para se obter uma lista sem uma hierarquia ou enumeração podemos usar a tag <ul> que adicionará um marcador por simbolo, esse tipo pode ser um circulo, ou quardado ou estrela ou outro marcador qualquer através do CSS.

  • HTML
  • CSS
  • JavaScript
  • SQL

O parâmetro 'type="X"' se aplica também às listas não ordenadas.

  • C
  • C++
  • C#
  • Java

Listas de Definições

A lista de definição possui duas tags a <dt> e a <dd> que estarão dentro da tag <dl>.

A tag <dl> (definition list), contém as tags <dt> (term definition) e abaixo teremos a tag <dd> (definição)
