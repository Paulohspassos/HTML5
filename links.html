Links Externos

O fundamento básico da linguagem de marcação HTML são os links ou âncoras e para isso usamos a tag <a>, para podermos acessar o endereço usamos um parâmetro chamado 'href="https://www.site-desejado.com/outros_parâmetros', além desse outro parâmetro muito utilizado é o 'target="_blank"' que abrirá o site em outra aba no navegardor permitindo assim que o usuário não perca o seu site de vista.

Exemplo de link/âncora Google

No exemplo acima o site do Google abrirá na mesma aba do seu site porém podemos abrir em outra aba utilizando o parâmetro 'target="_blank"' e isso é muito útil.

YouTube

Outro parâmetro importante é o 'rel="external"' que indicará ao navegador que se trata de um link para uma página externa

Links internos

Podemos também criar um link para outra página do nosso site para isso devemos adicionar o nome do arquivo, nesse caso em href entre as aspas duplas o nome do arquivo exemplo: <a href="pagina2.html" rel="next"> com isso o link abrirá a página 2 do nosso site.

Páginas em outras pastas

Para acessar uma página que esteja dentro de uma pasta diferente da pasta do documento principal, devemos configurar o parâmetro 'href="..."' da seguinte maneira, 'href="nome_da_pasta/nome_do_arquivo.html"'.

Caso o documento principal esteja dentro de uma pasta e seja necessário voltar uma pasta, então deveremos usar essa configuração, 'href="../nome_da_pasta/nome_do-documento.html"'.

Geralmente quando você faz um link para um artigo/página de terceiro usamos o parâmetro 'rel="nofollow"'.
