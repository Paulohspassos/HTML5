Imagens dinâmicas

Quando utilizamos imagens em documentos html, um dos desafios enfrentado pelos desenvolvedores é o dimensionamento dessas imagens para os diversos tamanhos de telas, para aproveitar melhor cada tela podemos ter imagens diferentes no documento para se adaptar melhor aos diversos dispositivos.

Para execultar essa funcionalidade de forma automatica usamos a tag <picture> ela permite várias funcionalidades das nossas imagens do documento.

Para se conseguir o resultado esperado devemos nos atentar a ordem das dimensões das imagens, ao escolher o tamanho da imagem base, por exemlo a pequena, ela deverá estar na tag <img src="imagem_pequena.formato"> na ultima linha dentro das tags <picture> e as imagens deverão obrigatoriamente acima na ordem decrescente, primeria linha <source> a imagem grande e abaixo, na outra tag a imagem pequena.
