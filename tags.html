Tags semânticas

Nas versões anteriores do HTML podiamos usar varios recursos de estilização e efeitos diversos para dar forma aos nossos sites porém devido a novas tecnologias inseridas em navegadores e servidores, intelencia artifical e outros, os códigos modernos precisam estar em conformidade com significados, semântica, para isso foram inseridas novas tags e delimitado a função das "partes" das páginas.

Antigamente, era comum utilizar a tag <div> com identificadores e classes para dividir a página em setores como cabeçalho, corpo principal, rodapé entre outros, hoje temos a disposição tags específicas para cada "setor" dos nossos documentos. e vamos falar uma pouco de algumas tags bem úteis.

<header>

A tag <header> é utilizada para definir um setor do código como sendo o cabeçalho da nossa página, assim fica mais fácil tanto para o desenvolvedor, usuários que tenham interesse e para os motores de busca entender a estrutura do documento e agilizar a busca de algum elemento dentro do código.

<main>

Essa tag é responsável por delimitar o setor principal da nossa página,local em que conterá os principais elementos do documento.

<aside>

Essa tag é utilizada para conter elementos secundários do documento como outros artigos, assuntos diversos, propagandas entre outros.

<footer>

A tag <footer> é a tag que marca o nosso rodapé onde são inseridas informações muito importantes como contatos e quem desenvolveu a página além de outras funcionalidades.

Outras tags

Temos muitas outras tags importantissimas para um desenvolvimento organizado dos nossos projetos em HTML5 e vamos tentar falar um pouco de cada um de alguns mais utilizados.

<nav>

A tag <nav> é utilizada para criar uma area de navegação com lista que pode levar o usuário a outras areas do nosso projeto, ou até mesmo disponibilizar outros setores da nossa página.

<hX>

As tags de títulos são as tags que têm a letra "h" seguida por um número que vai de 1 à 6, sendo a tag <h1> o título principal e os demais números seus sbtítulos, porém cada número representa um nível hierarquico abaixo do número menor, ou seja, 1 > 2 > 3 > 4 > 5 > 6, dessa forma o "h1" é o título principal de uma página e o "h6" seria o subtítulo menos importante da página.

O recomendado é que utilizem apenas um título por página, já os subtítulos podem ser usados quantas vezes forem necessário.

Podemos usar um título em "h1" em setores diferentes da nossa página como: um "h1" em uma tag "header", outro "h1" na tag "main", outro "h1" dentro de uma "section", "article", "footer". Porém é bom evitar mais de um "h1" dentro do mesmo setor da página.

<p>

A tag <p> serve para delimitar um parágrafo dentro do nosso documento e podemos ter quantas tags "p" forem necessarios.

<div>

A tag "div" é uma tag que serve para reservar um espaço na página que pode ser redimensionado conforme a necessidade do desenvolvedor

<section>

A tag "section" serve para estipular uma seção especifica no documento.

<article>

Serve principalmente para delimitar um artigo.

Essas são apenas algumas tags mais importantes no nosso documento, além dessas, o HTML possue uma infinidade de tags para as mais diversas necessidades e em todas existe uma aplicabilidade correta para cada uma, caso exista a necessidade de saber mais sobre essas que foram citadas aqui e outras que não foram citadas visite o site da w3schools.
